Windows的Claude code配置

2026-04-02 | 代码

Win+R输入cmd启动命令行：

npm install -g @anthropic-ai/claude-code

验证版本，成功会显示版本号：

claude --version

输入：

claude

一般会有错误：

Unable to connect to Anthropic services
    Failed to connect to api.anthropic.com: ERR_BAD_REQUEST
    Note: Claude Code might not be available in your country.

没挂全局代理导致的。

解决方法参考：
https://gist.github.com/docularxu/db0053008b9f41328f29d39ffcf7c2b2?permalink_comment_id=6014130

https://www.cnblogs.com/wzzkaifa/p/19079594

用Clash Verge，检查端口设置，我的是7897，然后在命令行：

set http_proxy=http://127.0.0.1:7897

现在可以进入Claude了，会显示：

Welcome to Claude Code v2.1.89
    Let's get started.

先让你选择theme，然后就会启动浏览器让你登录Claude账号，进入Claude console。
这里应该是让你认证账户。

我选择获取deepseekAPI，用下面的平台登录，在API keys里创建就好了：
https://platform.deepseek.com

需要永久配置的话，在系统变量里新建变量：
ANTHROPIC_BASE_URL=https://api.deepseek.com/anthropic
ANTHROPIC_AUTH_TOKEN=”自己的API密钥”
ANTHROPIC_MODEL=deepseek-chat
ANTHROPIC_SMALL_FAST_MODEL=deepseek-chat
